# Maintainers

This document lists the current maintainers of Misaka Cipher and their responsibilities.

## Core Maintainers

| Maintainer | GitHub | Role |
|------------|--------|------|
| Aethvion | [@Aethvion](https://github.com/Aethvion) | Lead Maintainer |

## Responsibilities

### Lead Maintainer
- Reviews and merges pull requests
- Triages and prioritizes issues
- Manages releases and versioning
- Sets project direction and roadmap

## Response Times

| Activity | Target Response Time |
|----------|---------------------|
| Issue triage | Within 3 business days |
| PR review | Within 5 business days |
| Security vulnerability | Within 48 hours |
| Question / discussion | Within 5 business days |

## Becoming a Maintainer

We welcome contributors who demonstrate consistent, high-quality involvement with the project. If you are interested in becoming a maintainer, please:

1. Contribute several meaningful pull requests
2. Participate actively in issue discussions
3. Help review other contributors' PRs
4. Reach out via [GitHub Discussions](https://github.com/Aethvion/Misaka-Cipher/discussions)

## Stepping Down

If a maintainer needs to step down, they should open an issue or discussion to transfer responsibilities.
